## File Descriptions

### Core Files

- **constants.py**: All physical constants, simulation parameters, and validation thresholds
- **run_all.py**: Main workflow script that orchestrates all simulations and analyses
- **requirements.txt**: Python package dependencies

### System Generation

- **create_bulk_system.py**: Creates FCC Al-5wt%Mg bulk system
- **create_dislocation_system.py**: Creates Al-5wt%Mg system with edge dislocation

### Simulation Scripts

- **run_bulk_diffusion.py**: Runs bulk diffusion simulations at multiple temperatures
- LAMMPS input files in `inputs/` directory

### Analysis Scripts

- **analyze_msd.py**: Extracts diffusivity from MSD data using Einstein relation
- **fit_arrhenius.py**: Fits Arrhenius equation and extracts D₀ and Q
- **analyze_dislocation_diffusion.py**: Analyzes pipe diffusion near dislocation cores
- **compute_binding_energy.py**: Computes Mg-dislocation binding energy E_b(r)
- **analyze_dsa.py**: Computes τ_diff and τ_wait to identify DSA regime
- **plot_results.py**: Generates all visualization plots
- **validate_simulation.py**: Validates simulation quality
- **generate_report.py**: Generates comprehensive markdown report

### Output Files

All analysis results are saved in `outputs/analysis/`:

- **CSV files**: Data tables for all computed quantities
- **PNG files**: Visualization plots
- **summary_report.md**: Final comprehensive report

## Workflow

1. **Setup**: Place potential file in `potentials/`
2. **Generate Systems**: Run `create_*_system.py` scripts
3. **Run Simulations**: Execute LAMMPS simulations (via scripts or manually)
4. **Analyze**: Run analysis scripts to extract diffusivity, fit Arrhenius, etc.
5. **DSA Analysis**: Compute DSA conditions using `analyze_dsa.py`
6. **Visualize**: Generate plots with `plot_results.py`
7. **Report**: Generate final report with `generate_report.py`

Or simply run: `python run_all.py`
